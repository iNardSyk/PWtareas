Colores

RGB, es la combinacion de los colores rojo, verde y azul

decimal: Este parrafo tiene un color rgb de (155, 25, 120)

hexadecimal: Este parrafo tiene un color rgb de #C67F12

porcentual: Este parrafo tiene un color rgb de (27%, 38%, 69%);

HSL, es la combinacion de matiz, saturacion y brillo

Transparente: En HSL, aun no se puede trabajar con transparecias como lo puede hacer HSLA, este parrafo tiene un color hsl de (0,50%,50%)

RGBA, funciona de la misma manera que RGB, pero añadiendo una variante de opacidad

Este parrafo tiene un color rgba de (155, 25, 120,.3)

HSLA, agrega una variable de opacidad al funcionamiento, desde 0 transparente hasta 1 opaco

Este parrafo tiene un color hsla de (0,50%,50%,.4)
